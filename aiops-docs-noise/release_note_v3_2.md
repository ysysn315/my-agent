# 发布说明 v3.2

本次版本重点是权限模型重构与前端主题切换，不涉及后端性能优化。

- 新增 12 个 RBAC 角色
- 登录页支持 SSO
- 前端路由懒加载优化
- 已知问题：夜间模式下图标偶发错位

回滚策略：
1. 回滚前端镜像
2. 保留数据库 schema
3. 重启 gateway

## 详细变更说明

### 权限模型重构
- 将原有的角色继承链拆分为显式授权列表
- 管理后台新增角色复制与模板导入能力
- 针对审计要求，新增权限变更审批日志

### 前端体验优化
- 菜单树懒加载策略改造，降低首屏包体积
- 登录页主题切换在暗色模式下支持自动记忆
- 设置页抽屉交互更新，减少误触发

### 数据库与接口兼容性
- 新增两张权限中间表用于角色绑定
- 历史接口保持兼容，旧字段保留两个版本周期
- 提供一次性数据迁移脚本，支持断点续跑

## 灰度与验证清单

1. 先灰度 10% 租户，观察 30 分钟
2. 检查登录成功率、权限校验失败率、页面加载耗时
3. 若权限拒绝率异常上升，立即切换回旧鉴权链路
4. 灰度完成后执行全量发布并做抽样回归

## 常见问题（非性能根因）

- “用户看不到菜单”通常是角色绑定遗漏
- “按钮置灰”通常是权限点命名不一致
- “页面闪烁”通常与前端缓存失效策略有关

注：本文属于版本发布记录，不用于 CPU、内存、慢查询等故障根因判断。
