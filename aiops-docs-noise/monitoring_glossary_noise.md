# 监控指标术语手册（噪声词典）

> 说明：这是术语解释，不是排障方案。

## 指标术语

- CPU Usage：CPU 使用率
- Memory RSS：常驻内存
- Heap Used：JVM 堆内存占用
- Full GC Count：Full GC 次数
- P95/P99：延迟分位数
- Error Rate：错误率
- QPS：每秒请求数

## 常见关键词解释

### “CPU 高”
可能是负载升高，也可能是压测、批处理或计划任务，术语本身不等于故障。

### “OOM”
表示进程内存不足事件，出现该词并不直接说明是内存泄漏。

### “Slow Query”
表示 SQL 超过阈值，可能与索引、锁等待、慢磁盘、网络抖动相关。

### “500 错误”
是 HTTP 状态码，代表服务器侧错误，不直接等于数据库故障。

## 观察建议（非根因）

1. 先看趋势，再看瞬时值
2. 先看相关指标，再看单点指标
3. 先确认时间对齐，再分析因果

注：本文不提供告警处置步骤，主要用于术语统一和培训。
